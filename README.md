Fizz Buzz をすべての言語でコーディング。


# 問題
- 1から100までの数について、以下の操作を行う。
  - 3で割り切れるなら「Fizz」と出力。
  - 5で割り切れるなら「Buzz」と出力。
  - どちらでも割り切れるなら「Fizz」と出力。
  - 以上を満たさなければその数を出力。


# 注意事項
- 以上の操作をする返り値`void`の関数「FizzBuzz」を作成し、存在する場合は`main`関数内で実行する。
- ワンライナーなどの制限は設けず、最も標準的と思われる回答をする。
- 外部からファイルを読み込んだりはしない。
