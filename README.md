Fizz Buzz をすべての言語でコーディング。


# 問題
- 1から100までの数について、以下の操作を行う。
  - 3で割り切れるなら「Fizz」と出力。
  - 5で割り切れるなら「Buzz」と出力。
  - どちらでも割り切れるなら「Fizz」と出力。
  - 以上を満たさなければその数を出力。
- それぞれの出力について最後に改行し、数字は`int`型で出力する。


# 注意事項

- 外部からファイルを読み込んだりはしない。
